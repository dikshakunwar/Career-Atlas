# 🌍 CareerAtlas

CareerAtlas is a full-stack web application that connects students with job and internship opportunities through an interactive map-based interface. The platform allows recruiters to post opportunities while enabling students to discover, search, and bookmark openings based on location.

## 🚀 Overview

CareerAtlas simplifies the job search process by combining an intuitive user experience with location-based visualization. Users can explore opportunities across different regions, while recruiters can efficiently manage job postings through a dedicated dashboard.

## ✨ Features

### 👨‍🎓 Student Features

- 🔐 Secure authentication and authorization
- 🗺️ Interactive Mapbox-powered job map
- 🔍 Smart search and filtering
- 📌 Bookmark favorite jobs
- 📄 View detailed job descriptions
- 👤 Personalized user dashboard

### 👨‍💼 Recruiter Features

- 🔐 Recruiter authentication
- ➕ Create and publish job or internship listings
- ✏️ Update and manage posted jobs
- ❌ Delete job listings
- 📊 Dashboard for managing opportunities
